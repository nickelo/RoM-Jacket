Arcade Machines

ARCADE SYSTEMS Arcade Machines Defaults to MAME. MAME romsets are zip files which RoM-Jacket groups into a hidden "SOURCE" folder inside the main folder eg: ..\Games\Console\Arcade\SOURCE" All identified games are aliased into the properly named folders using MAME's database. Accepts: .zip CPS (Capcom Play System) Defaults to Final Burn Alpha. All ROMs are placed in a hidden directory within the "CPS" folder called SOURCE. Though these ROMs use the same naming convention as MAME games, each game must contain all ROMs required. The ROMs are identified and a launcher is created inside each game's properly listed name. Accepts: .zip CPS II Defaults to Final Burn Alpha. All ROMs are placed in a hidden directory within the "CPS 2" folder called SOURCE. Though these ROMs use the same naming convention as MAME games, each game must contain all ROMs required. The ROMs are identified and a launcher is created inside each game's properly listed name. Accepts: .zip CPS III Defaults to cps3emulator All ROMs are placed in a hidden directory within the "CPS 3" folder called SOURCE. These are not the CHD ROMs used by MAME. Also, though these ROMs use the same naming convention as MAME games, each game must contain all ROMs required. The ROMs are identified and a launcher is created inside each game's properly listed name. Accepts: .zip NeoGeo MVS Defaults to Final Burn Alpha. All ROMs are placed in a hidden directory within the "Neo Geo MVS" folder called SOURCE. Though these ROMs use the same naming convention as MAME games, each game must contain all ROMs required. Requires a BIOS file. The ROMs are identified and a launcher is created inside each game's properly listed name. Accepts: .zip Sega Model 2 Defaults to m2emulator. All ROMs are placed in a hidden directory within the "Sega Model 2" folder called SOURCE. The ROMs are identified and a launcher is created inside each game's properly listed name. Accepts: .zip Taito Type X2 Defaults to ArcadePCLoader. These games are complete Windows applications and each has a corresponding (short name) folder inside ArcadePCLoader's "Games" directory. RoM-Jacket will create shortcuts to the executables created by ArcadePCLoader.
