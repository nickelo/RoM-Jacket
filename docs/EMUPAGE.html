Emulators


This menu will allow you to install emulators.

All Recommended

Selecting "All Recommended" will download and install the recommended and supported emulators listed. Each supported system has an emulator which can be configured by RoM-Jacket. Many other emulators are available for installation and some systems offer wizards for alternate emulators.

Caveats:
Emulators should be installed via this utility as RoM-Jacket installs "Helper" programs in tandom to assist in ROM-launching automation and stability.

RetroArch

This emulator is fully integrated into RoM-Jacket. All of RoM-Jacket's supported consoles compatible with RetroArch are configurable for use with it.

Mess/UME

Mess BIOS files should be installed in the emulator's "roms" directory.
Defaults to C:\Emulators\Mess\roms

Unsupported consoles follow the custom console paradigm using the Mess command line syntax, eg:
systemname -mediatype

RoM-Jacket's supported consoles compatible with Mess/UME are preconfigured for use.

Drivers


This menu item allows you to select dependencies to install

All Dependencies

This will install DirectX and all Visual C++ dependencies.

Daemon Tools

This menu allows you to configure the disk-mounting program "Daemon Tools" The Daemon Tools program is needed for the SSF Sega Saturn emulator, the UNZ FM-Towns emulator and may be required for any emulator which cannot directly read cd/dvd image files.
!!!!IMPORTANT!!!!
Open the Daemon Tools program.
If you do not have a SCSI drive in the list, add a SCSI drive.
Right click on the new drive icon.
select Device Paramaters

Uncheck Auto insert notification

 

Visual C++

Many emulators require these runtimes

DirectX

Installs and Updates DirectX libraries

Joystick Drivers

RoM-Jacket preconfigures emulators and joystick profiles (XPadder and antiMicro) for the XBox 360 button-number layout.

XBox360 Drivers

This installs the XBox 360 Joystick drivers from Microsoft. These are not needed by Windows 8x machines. Install these if you're not using Windows XP or Windows 8x.

SCP Wrapper for Dualshock Joysticks

This installs the SCP Wrapper for Sony Dualshock joysticks.

This is the most reliable and easiest way for DS3/4 users to use their joysticks in Windows 7/8x.

MotioninJoy (PS3 Six-axis) -offline version-

This installs the MotionInJoy driver for Sony PS3 Six-axis joysticks. This is a good alternative to the SCP Wrapper. MotionInJoy users should use the "XBOX 360 Controller Emulator" option for best results. RoM-Jacket will install the setup files, however the configuration for motioninjoy is left to the user.

DS4Windows

DS4Windows is a Sony DualShock 4 driver for windows. It is installed however the configuration of this driver is left to the user.

Better DS3

This installs the Better DS3 program which is a simple and easy interface to use for the MotioninJoy Drivers.

 

Caveats: Many of Rom-Jacket's supported emulators leverage "Helper" programs to assist in automating the launching of ROMs

BIOS

This menu item will allow you to download and install bios files.

Install

This menu allows you to select a file or archive containing bios files.  RoM-Jacket can detect BIOS files by name and via a HASH check. Files and archives may also be added by dragging and dropping the file or archive onto the "Bios-Folder" icon located on your desktop.

Download

List

s This menu allows you to select a community list containing links to bios files and archives.

Torrent

This menu allows you to provide a torrent file. Files provided by a torrent will be downloaded to your bios folder.

URL

This menu allows you to provide a URL You can specify a torrent, magnet, ftp or http URL. Files will be downloaded to your bios folder.

 

The Paradigm:
Archives are exploded into the "Bios-Folder."
Recognized BIOS files found are copied to corresponding emulator's BIOS-folders. Caveats:
Installing emulator BIOS files must be done prior to configuring the console.

ROMs

This menu has downloading options for RoM-Jacket. All files are downloaded to the RJ-Sort-Folder.

Torrent

RoM-Jacket can download files using a .torrent file.

URL

You can enter the URL of a magnet link or any other file.
