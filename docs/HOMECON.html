Defaults to 4DO. Requires a BIOS file.

Accepts:
.iso
.bin
.cue
.nrg
.mdf
.cdi

Caveats:
4DO uses a file "JohnnyInputBindings.xml" to define joystick controls.
To use direct/xinput for this emulator, reconfigure your joystick/s using tthe 4DO gui and close the application. This file can then be propagated throughout your library.

Nintendo 64
Defaults to Project 64.

Accepts:
.n64
.zip
.usa
.pal
.*64

Caveats:
The Project64 NRage plugin uses a separate NRage.ini to store input bindings. Unique joystick device-id numbers are stored in the Project64.cfg file for Jabo's Input plugin as well. In the Utilities section you can use the N64Joystick option in "File" menu to propagate your Joystick settings throughout your library if you decide to use Xinput.
