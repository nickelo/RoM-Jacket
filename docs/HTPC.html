Frontend Considerations


RoM-Jacket's frontend MetaData should be downloaded and installed prior to frontend configuration

The Paradigm:
XBMC/KODI is a good comprimise for users who are looking to build a RoM-Jacket library and want to download box-art with other assets.
Between Advanced Launcher and Rom Collection Browser, XBMC/KODI is a fast and functional browser for most systems

What do I want and which option is best for me?

Users of RoM-Jacket have different needs and so several HTPC frontend options are available.
Artwork such as box-covers, banners and backgrounds can be scraped to your ROM jackets via some frontends however many enthusiasts may wish to import or transfer existing assets.

Advanced Users

Advanced users have artwork migration tools available for each frontend. A robust set of configuration options for Hyperspin enable managing and creating a configuration easy.

Enthusiasts

Emulationstation, Advanced Launcher and Rom Collection Browser plugins scrape artwork and metadata for games from the internet.

Impulsive and recreational players

Mediabrowser, retroFE and Emulationstation will populate any games added to a library dynamically.

Deployments

Cabrio-FE, retroFE and Emulationstation are ideal frontends for older computers or lightweight libraries.

PC Gamers

Steam users can browse their libraries easily.

Windows™ Users

Oblytile and Mirror+ are great for Windows™ users who don't want the overhead of an HTPC frontend, but would like to be able browse using box-art icons for their games.

Caveats:
Windows 8™ users who wish to use Mediabrowser must install Windows MediaCenter™.
Oblytile is only available for Windows 8™ users
