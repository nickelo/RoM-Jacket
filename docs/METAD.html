MetaData

All artwork and metadata can be downloaded from MEGA which is much faster than using RoM-Jacket's repositories.

Download, install and convert artwork, icons, logos, videos and full-hd backgrounds for your Frontend.

Download/Install

This menu will show the download status of banners, backdrops, icons and the number of videos.

Icons

High quality photographic icons in PNG format are ready to be downloaded and used by your frontend.

Backdrops

RoM-Jacket's repository includes Full HD background images for over 80 consoles.

Logos

RoM-Jacket's repository includes high quality custom Logos (Banners/Wheel-art) in PNG format.

Video

This will bring up a menu that will allow you to download all videos for your existing consoles, or select from a list which console-video to download.
Videos are downloaded to a hidden folder "backdrops" in the root of the console's directory with a corresponding filename in flash video format (FLV), or MP4.
These videos can be used by Mediabrowser/Gamebrowser, Cabrio-FE and Hyperspin.

 

Current

This will allow you to download only the icons, logos, backdrops and videos for consoles in your current library.

Convert

This will rename artwork scraped by Advanced Launcher to the standard naming convention.

...\Console Name\Game Name\Game Name_Boxart.jpg ==...\Console Name\Game Name\Folder.jpg
...\Console Name\Game Name\Game Name_Fanart.jpg ==...\Console Name\Game Name\Backdrop.jpg

Alias++

This is for users who wish to create a set of mirrors which are divided by console type:
Computers/Handhelds/Home Consoles/Arcade Machines

 

The Paradigm:
Information and artwork collected (scraped) by front-ends is reprocessed by RoM-Jacket in order that it may be shared across them.
All data is stored in the corresponding ROM's folder.
Each time you add ROMs and scrape metadata or collect artwork the front-ends should be reconfigured to allow the data to populate properly.
Caveats:
Videos are approximately 10mb each.
Downloading all videos for all supported consoles can exceed 600mb.
If you have previously installed a supported front-end, the existing configuration files are backed up as renamed files
(*.sbak and *.orig).
