retroFE

RoM-Jacket's frontend MetaData should be downloaded and installed prior to frontend configuration


GENERAL USAGE

Follow these steps to create a retroFE setup.

1. Re/Install: retroFE.
2. Create Mirrors: for your consoles.
3. Add: systems to your configuration.
4. Import: art from your romjackets or Hyperspin media

CONFIGURATION

The Paradigm:
retroFE uses a name-matching system to associate assets in a similar way to Hyperspin.

Mirror

This will create a shortcut file (mirror) for each launcher in selected consoles.

ADD

Adds consoles to the menu.xml in the Main collection, and copies ROMs into retroFE
"...\collections\Console Name\roms" directory.

Setup

Redownload, set the location of an installation or reinstall retroFE.

Artwork/Media

Import/Export

This will transfer Artwork/Video to or from your Rom-Jackets or Hyperspin Library to or from your retroFE Installation.
Box-covers, box-back, media-front, backdrops, logos and video are supported at this time.

Theme

The RJ theme is the only theme currently supported.
