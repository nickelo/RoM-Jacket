Manually adding ROMs to your Library

If you have already separated your machines by the proper names you can move these folders inside the "Console" folder:

defaults to "C:\Games\Console\Console Name"
Be certain only ROMs are in a console's folder: Any file will be jacketized when configuring the console.

SORT

This menu item is used to add ROMs from the "RJ-Sort-Folder" to your library.

Extensions & Databases

Hash vs Wizard

During the sorting process, ROMs are moved from the RJ-Sort-Folder to the corresponding console's folder. Two methods are provided to assist in the sorting process.

HASH

ROM databases sort the contents of your RJ-Sort-Folder.

Paradigm:
Known files are renamed to the indexed name and then
 moved and jacketized according to the associated Console system.
Caveats:
Searching for all systems in your RJ-Sort-Folder may be slow depending on the number of items in the RJ-Sort-Folder.
Archives are NOT extracted.

WIZARD

Identified ROMs inside the RJ-Sort Folder are automatically moved into their corresponding library's console folder.
ROMs found within archives such as .zip, .rar or .7z are extracted into the appropriate console folder and the archive is moved into the "SORTED ARCHIVES" directory inside the RJ-Sort-Folder.

Paradigm:
All archives are indexed and ROMs containing extensions exclusive to a console or emulator are moved to their associated libraries. Caveats:
Many libraries consist of ROMs utilizing the same extension:
eg: .bin|.cue/.bin|.rom

Custom/Unsupported Consoles

When importing ROMs into your library several methods can be used to identify consoles.

Import

Systems are imported as they are and do not attempt to identify them..

Detect

RoM-Jacket uses simple name-searches and detected systems are either merged or renamed to the detected system type.

Ignore

Ignore the unidentified consoles and continue sorting.

Match

Match an unidentified console (folder) to a supported console.
You can select a supported/recognized system and rename the unidentified system.

Importing

The following options provide additional ways to migrate existing libraries and configurations into your library.

Archive

This menu allows you to select a single archive to explode into your RJ-Sort-Folder.

Folder

This menu allows you to select a folder (which may contain archives) to expand into your RJ-Sort-Folder.

Migrate

This menu allows you to select an existing library and add its items to yours.  RoM-Jacket guesses based on the selected library's console's names.  Select the folder containing the console directories.>
