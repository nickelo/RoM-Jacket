Getting Started


Initial settings define global computer settings. RoM-Jacket will attempt to detect and allow you to install Xpadder and Daemon Tools.

Three icons will appear on the desktop after selecting your library locations:


RJ-Configurator

This tool configures Rom-Jacket
This can be found on the desktop and also in the RJ installation folder. In addition to being the configuration tool, dragging ROMs onto this desktop-icon will allow you to copy/move ROMs to your RJ-Sort Folder and begin the sorting process
Launching the Rom-Jacket Configurator from the installation folder will enable logging.


RJ-Sort Folder

This is the RJ-Sort-Folder where ROMs are sorted.
This folder will be in the same location you selected for your game library.All ROMs you wish to automatically identify and sort to your library will be processed in this folder. A sub-folder called "SORTED ARCHIVES" can also be found in this folder.
Defaults to C:\Rom-Jacket

RJ-BIOS-Folder

This tool sorts BIOS-files.
Drag and drop bios files you wish to install onto this icon. Files are sorted in the BIOS folder of the RoM-Jacket installation
Defaults to ( ..\RJ\net\BIOS )

Initialize-Rom-Jacket

This icon can be found in the installation folder and it will delete all desktop/explorer icons created by RoM-Jacket and reset all program settings.


In addition to the desktop icons, three icons can also be found in the favorites section of Windows Explorer on Windows 7 and 8.


Consoles

This is a shortcut to your Console Library (defaults to C:\Games\Console)


Emulators

This is a shortcut to your emulators (defaults to C:\Emulators)


Linux users:

RoMJacket can be run through WINE


For Ubuntu/Debian based systems: open the terminal and enter these commands:
sudo apt-get install wine winetricks
export WINEDLLOVERRIDES=mscoree,mshtml=
winetricks -q wsh56 wsh56vb mfc42
wine ~/Downloads/RJ.exe /S


Use these commands to create a Desktop Shortcut
echo "wineconsole cscript C:\\\RJ\\\RomJacket\\\BEGIN.vbs" >> "Rom-Jacket-Configurator.sh"
mv Rom-Jacket-Configurator.sh ~/Desktop/
sudo chmod +x ~/Desktop/Rom-Jacket-Configurator.sh

After initializing RoMJacket you should be able to execute the "Rom-Jacket-Configurator.sh" tool on your desktop to launch future sessions of RoMJacket.
Drag and drop is not supported under Linux at this time

General Usage

The following steps should be taken when adding any new consoles.

STEP 1: Install emulator software.

All supported emulators can be downloaded & installed simultaniously or individually.(video)

STEP 2: Install BIOS-files.

Download, import or drag'n drop bios files onto the "Bios-Folder" to install them automatically.(video)

STEP 3: Add ROMs to your Library

Download, import or drag'n drop ROM libraries & ROM archives.(video)

STEP 4: Jacketize.

The default options can be chosen by simply pressing the enter key in any wizard
A wizard sets up options for a library or all libraries.(video)

STEP 5: Deploy a frontend.

Select and play your games using your joystick.(video) It is highly recommended to download and install RoM-Jacket's artwork assets before deploying.
