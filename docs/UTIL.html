Utilities

Migrate

If you have a RoM-Jacket ROM-Library you wish to import to your existing drive/location, you can use this utility to automatically migrate existing settings to your current location or optionally set the location of the any library. All supported emulators and systems can be migrated.

This can be used to replace any string with any other string.
^"&%>< characters must be escaped

Copy Cfg

This menu allows you to propagate settings from one file throughout an entire console's library.
RomJacket supports exclusive audio/video and input (joystick) settings for *most* of the default emulators
If a launcher is selected (.bat) RoM-Jacket can propagate the file throughout a console's library and update or replace all other launchers.
Any file can be propagated (eg: an xpadder profile) and subdirectories are supported.

Cloud

This menu allows you to backup your save files and save-states into a cloud. Google Drive, Microsoft OneDrive and Dropbox will be automatically detected.
This is ideal for users who want to share saves or play from multiple computers.

Execute

This menu allows you to select a ROM or an entire console's library to transform into a portable executable. The ROM, any additional files specified by the user, an emulator and special configuration files are compiled and saved to the location of your designation.
If boxart is found, it will be used as the icon for the executable and shortcut it creates if installed..

 

Caveats:
Adding additional files to a ROM or ROM-set executable must be added one at a time.
Only the default emulators are supported for exe creation at this time.
