Steam

RoM-Jacket's frontend MetaData should be downloaded and installed prior to frontend configuration

If it is not detected, this menu will prompt you to locate Steam.
Steam uses RoM-Jacket's mirrors (defaults to C:\Games\Mirrors) to build the library.

Backup Shortcut

RoM-Jacket wil attempt to detect your shortcuts.vdf file. A backup named shortcuts.vdf.orig will made.

Mirrors

You may choose to create mirrors of your RoM-Jacket Library for Steam to use.

 

Ice vs. RoM-Jacket

RoM-Jacket can use a 3rd party utility (Ice) or build a configuration file using original code.

Ice

Ice can scrape metadata and art, but the library requires a unique naming convention at this time. Ice is quick to setup, and easy to update

RJ

RoM-Jacket can build its own shortcuts.vdf file that allows for non-standard characters in game titles. This option may take a few minutes to setup depending on the size of your library.

 

Caveats:
Games with titles containing non-standard characters will cause Ice to fail
RoM-Jacket's current Library naming conventions break the scraping api of Ice.
To enable scraping, a seperate library location containing only systems with the supported naming conventions must be used.
