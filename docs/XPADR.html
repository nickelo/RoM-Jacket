XPadder


Xpadder can be used to map all major functions of each emulator Install XPadder to a location of your choice.

You can get the latest version here.

Setup

Before you setup XPadder,be sure to install the appropriate drivers.


Opening the XPadder.exe you are presented with a similar dialoge box:

 

Joystick.xpaddercontroller

RoM-Jacket is packaged with several XPadder files pre-configured for 2 XBox360 Joysticks.
Each Joystick identified by XPadder should be assigned an ":xpaddercontroller" file. A square will appear in the top-panel for each identified joystick.
Joy1.xpaddercontroller and Joy2.xpaddercontroller files are hardware presets for XBox360 Joysticks. If you are using a controller not compatible with Microsoft's XBox360 Drivers, you can select ''New...'' from the dropdown menu to setup and create an xpaddercontroller file for your controller.
Once you have assigned a xpaddercontroller file to your joysticks XPadder will automatically load these xpaddercontroller files whenever these joysticks are plugged in.


Once you have assigned your joysticks "xpaddercontroller" files you can assign keys:

 

Joystick.xpadderprofile

In order to control your frontend with a joystick, XPadder is needed. The default profile (MediaCenter.xpadderprofile) is provided to control most frontends effectively.

How it works
Each time you launch a game, the profiles for that game load.
When you exit the game, the Frontend profile is re-loaded.

Initialize

Extracts the preconfigured XPadder files to your XPadder location.

Re/Configure

Creates or recreates profiles for each game.

Profile Types

Keyboard

These profiles contain keyboard mappings for emulators in their factory state or commonly used keys. If you do not plan on using a joystick or have a joystick which is incompatible with RoM-Jacket's xpadderprofile presets, you can try this option.

Blank

These profiles are completely blank. This is useful for users who wish to use global emulator configurations but want to use XPadder only to control a frontend.

Joy

Many emulators auto-detect joysticks and assign buttons automatically. These profiles contain mappings for emulator functionality (save/load state, reset emulator, save snapshot etc...) only. Users with joysticks which have XBox360 compatible drivers OR joysticks with identical button number layouts should use this option.

The Paradigm:
RoM-Jacket's Xpadder profiles have been designed for the XBOX 360 Joystick, however many joysticks seem to work well them.

REVERT

Toggles the annoying dialoge box Windows UAC (User account control) generates when using RoM-Jacket's Xpadder on Windows 7/8x-.
